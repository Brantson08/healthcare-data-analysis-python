# Healthcare Data Analysis (Python)

## Overview
This project demonstrates an end-to-end data analysis workflow using Python, including:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualisation of trends and patterns
- Insight generation for decision support

The analysis reflects tasks commonly performed in healthcare and public-sector environments.

## Objectives
- Identify patterns in patient data
- Detect trends that may impact operational performance
- Demonstrate structured analytical workflow using Python

## Tools & Technologies
- Python
- pandas
- numpy
- matplotlib
- Jupyter Notebook
